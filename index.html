Que es HTML?

HTML, acrónimo en inglés de HyperText Markup Language ('lenguaje de marcado de hipertexto'), hace referencia al lenguaje de marcado utilizado en la creación de páginas web. Este estándar que sirve de referencia del software que interactúa con la elaboración de páginas web en sus diferentes versiones. Define una estructura básica y un código (denominado código HTML) para la presentación de contenido de una página web, que incluye texto, imágenes, videos, juegos, entre otros elementos. Este estándar es gestionado por el World Wide Web Consortium (W3C) o Consorcio WWW, una organización dedicada a la estandarización de la mayoría de las tecnologías asociadas a la web, especialmente en lo relacionado con su escritura e interpretación. HTML se considera el lenguaje web más importante y su invención crucial para el surgimiento, desarrollo y expansión de la World Wide Web (WWW). Es el estándar que prevalece en la visualización de páginas web y es adoptado por todos los navegadores actuales.


Historia de HTML

Su historia se remonta a 1989, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear), lo inventó para compartir artículos científicos:

  • 1989: Tim Berners-Lee inventa el HTML para compartir artículos científicos.
  • 1991: Se publica el primer documento formal sobre HTML, titulado “Etiquetas HTML”,
  • 1993: Se publica la primera versión de HTML, que era una variante del Lenguaje de Marcado General Estándar (SGML).
  • 1995: Se publica la versión 3.0 del estándar HTML.
  • 1997: Se publica la versión 4.0 del HTML, recomendada por la W3C.
  • 1999: Se publica la versión 4.01 del HTML, que se convierte en estándar oficial.
  • 2006: Se publica la versión 5.0 del HTML, que introdujo nuevas etiquetas semánticas, soporte para multimedia y mejor interacción con aplicaciones web.

Editores de HTML

Algunos editores de HTML son:

  • Brackets: Un editor gratuito de Adobe que se centra en el diseño web visual. Cuenta con una vista previa en tiempo real, una interfaz sencilla y moderna, y soporte para extensiones.
  • Komodo: Un editor comercial que ofrece una experiencia de desarrollo completa y eficiente. Permite trabajar con múltiples lenguajes de programación y tecnologías web.
  • Adobe Dreamweaver: Un software de diseño web especializado para crear, abrir y editar formatos HTML.
  • Setka: Una herramienta diseñada para personas que están aprendiendo a codificar. Permite editar el código HTML de forma colaborativa en tiempo real.
  • Notepad++: Un editor de texto que se utiliza para el desarrollo web.
  • Sublime Text: Un editor de texto que se utiliza para el desarrollo web.

Tu primer documento

Para crear un documento HTML, puedes seguir estos pasos:

  1. Abrir un programa de edición de texto:
    • En Windows, puedes usar el Bloc de notas
    • En Mac, puedes usar TextEdit
    • En LibreOffice, puedes usar Writer/Web
  2. Crear un código HTML:\
    • En el Bloc de notas, puedes escribir el texto, etiquetas y espacios
    • En TextEdit, puedes introducir el código HTML
    • En LibreOffice, puedes escribir o pegar texto, imágenes, tablas y otros objetos
  3. Guardar el archivo con la extensión .html:
    • En el Bloc de notas, puedes seleccionar Archivo > Guardar como
    • En TextEdit, puedes seleccionar Archivo > Guardar
    • En LibreOffice, puedes guardar el documento como Documento HTML

Un documento HTML comienza con la etiqueta (html) y termina con (/html). Dentro del documento, hay dos zonas principales: el encabezamiento y el cuerpo.


Que son etiquetas HTML?

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo. Es decir, las etiquetas HTML son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML


Que son atributos en HTML

Los atributos en HTML son fragmentos de código que se utilizan para modificar el comportamiento o la visualización de los elementos HTML. Se agregan a las etiquetas HTML utilizando la sintaxis "nombre de atributo=valor". Existen varios tipos de atributos HTML, como los requeridos, los opcionales, los estándares y los de evento.


Reglas fundamentales de HTML

Algunas reglas fundamentales del HTML son:

  • La estructura básica de un documento HTML comienza con (!DOCTYPE html).
  • La etiqueta (html) envuelve todo el contenido de la página.
  • Dentro de la etiqueta (html) se encuentran las etiquetas (head) y (body).
  • La etiqueta (head) contiene metadatos y enlaces a hojas de estilo o scripts.
  • La etiqueta (body) alberga el contenido visible de la página, como texto, imágenes y enlaces.
  • Las etiquetas HTML no distinguen entre mayúsculas y minúsculas, pero es una buena práctica escribirlas en minúsculas.
  • Los atributos se incluyen en la etiqueta de apertura de un elemento, nunca en la de cierre.
  • Los archivos HTML se identifican por la extensión .html o .htm.

Etiquetas para hacer un documento html

Para crear un documento HTML, se utilizan las siguientes etiquetas:

  • (!DOCTYPE html): Define el tipo de documento
  • (html): Inicia el documento
  • (head): Contiene información general del documento, como el título de la página, metaetiquetas, scripts y estilos
  • (title) (/title): Inserta el título del documento
  • (body) (/body): Señala el cuerpo del documento
  • (h1) … (h6): Muestra títulos y subtítulos
  • (div): Define un bloque de contenido o sección de la página web
  • (a) (/a): Crea enlaces hacia documentos externos o internos
  • (strong): Define texto importante, aplicando negrita
  • (br): Realiza un salto de línea
  • (p) (/p): Representa un párrafo
  • (img) (/img): Coloca una imagen en el documento
  • (table) (/table): Define una tabla

Estas serian unas cuantas etiquetas mas utilizadas.
