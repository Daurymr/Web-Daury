Que es CSS?

CSS (Cascading Style Sheets) es un lenguaje de programación que permite controlar el diseño y presentación de las páginas web. Se utiliza para dar estilo a los elementos HTML, como el color, la fuente, la posición, las animaciones y los efectos visuales. El CSS funciona junto con el HTML, que se encarga del contenido básico de los sitios. El CSS separa el contenido del diseño, lo que facilita la gestión del aspecto visual de las páginas web.

Algunas caracteristicas son:

  • Se basa en reglas, donde cada regla se abre con un selector y se cierra con un conjunto de llaves.
  • Las características se aplican de arriba a abajo mediante reglas que poseen un esquema prioritario.
  • Permite agrupar reglas que aplican a diferentes etiquetas en una sola regla con un selector múltiple.
  • Permite utilizar selectores de etiqueta, clase, ID, combinadores y pseudo-clases.

Como utilizar CSS con HTML?

CSS permite separar el contenido HTML de su presentación visual. Es decir, te permite mantener el código HTML limpio y estructurado, mientras que el estilo se define en un archivo CSS separado. Este orden mejora la legibilidad del código y facilita el mantenimiento, así como la actualización de los estilos.

Para incluir CSS en tu documento HTML se puede incluir una regla de enlace en el elemento HEAD del documento. Tambien se puede utilizar el atributo de estilo (style) en el elemento HTML para especificar los estilos de esa clase o contenedor.


CSS en linea

El CSS en línea, o inline styles, es una técnica para definir un elemento único en un documento HTML con hojas de estilo insertadas. Se trata de una forma de añadir estilos en CSS que consiste en escribir el CSS directamente en la etiqueta HTML a través del atributo style.


CSS interno.

El CSS interno es una forma de agregar estilos a una página web HTML, incrustando el código CSS en el mismo documento. El CSS interno se diferencia de externo, de manera que se introduce un atributo (style) para darle estilo a la pagina directamente desde el documento HTML.


CSS externo

El CSS externo es un archivo independiente que se vincula a una página web para definir su estilo:

  • Se crea con un editor de texto, como Notepad++
  • Se enlaza a la pagina web mediante una etiqueta (link) en la seccion de encabezado del documento HTML.
  • Permite que varias paginas web utilicen el mismo archivo CSS.

El CSS externo es un método eficiente para estilizar sitios web grandes, ya que permite cambiar el estilo de todo el sitio con un solo archivo. Además, al separar el código CSS de los archivos HTML, estos últimos tienen una estructura más limpia y un tamaño menor.
