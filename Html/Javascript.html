Que es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para hacer que las páginas web sean interactivas y dinámicas:

  • Permite crear contenido de actualizacion dinamica.
  • Controla multimedia.
  • Anima imagenes.
  • Valida formularios.
  • Actualiza fuentes de redes sociales.
  • Muestra mapas interactivos.

JavaScript fue creado en 1995 por el programador norteamericano Brendan Eich. Se diferencia de Java, ya que son dos lenguajes diferentes con propósitos muy distintos.


Como utilizar JavaScript con HTML?

Hay dos formas de insertar JavaScript en HTML y hacer que funcionen juntos. Echemos un vistazo a algunas de las formas en que podemos conectar JavaScript con HTML.

La primera forma de insertar JavaScript en HTML es directa. Puedes hacerlo utilizando la etiqueta script, que debe envolver todo el código JavaScript que escribas

JavaScript en linea

El codigo JavaScript se puede introducir en linea en un documento HTML utilizando la etiqueta HTML dedicada (script). Esta etiqueta envuelve el codigo JS. La etiqueta script puede colocarse en la seccion head del documento o en la seccion de body.


JavaScript en archivo separado

A veces, insertar JavaScript en HTML directamente no parece ser la mejor opción. Sobre todo porque algunos scripts de JS deben utilizarse en varias páginas, por lo tanto, es mejor mantener el código JavaScript en archivos separados.

Es por esto que la forma más aceptable de incluir JavaScript en HTML es importando archivos externos. Se puede hacer referencia a estos archivos desde adentro de los documentos HTML, al igual que hacemos referencia a los documentos CSS.
