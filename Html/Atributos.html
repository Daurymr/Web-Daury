Atributos Globales

Los atributos globales son atributos comunes a todos los elementos HTML; se pueden usar en todos los elementos, aunque es posible que no tengan ningún efecto en algunos elementos. Los atributos globales se pueden especificar en todos los elementos HTML, incluso aquellos no especificados en el estándar.

Los atributos globales se utilizan para introducir parámetros adicionales a todos los elementos HTML. Por lo tanto, son atributos que permiten modificar características específicas, como de diseño o funcionalidad, de cualquier elemento HTML, ya sea un párrafo, un encabezado, un enlace o una imagen.


Class

El atributo class en HTML es una lista de clases que identifica un elemento HTML. Define el nombre de una clase o una lista de clases para aplicar estilos en cascada (CSS). El atributo class en HTML es una lista de clases que identifica un elemento HTML.


ID

El atributo id en HTML es un identificador único que se utiliza para identificar un elemento en un documento HTML.

Este atributo se puede utilizar para:

  • Vincular el elemnto en scritps.
  • Vicular el elemento en hojas de estilos (CSS).
  • Dar formato a la vista de operador con CSS.
  • Manipular el modelo de objeto de documento con codigo JavaScript y DHTML.


Title

El atributo title en HTML es un bloque de información adicional que se puede adjuntar a un elemento, como un enlace o una imagen, para proporcionar información adicional al usuario. Esta información se muestra como un texto de información sobre herramientas cuando el mouse se mueve sobre el elemento. El atributo title se puede usar en cualquier elemento HTML.

Algunos de sus usos típicos son:

  • En un vínculo, para mostrar el título o la descripción del documento vinculado.
  • En un elemento mediático, como una imagen, para mostrar una descripción o créditos asociados.
  • En un párrafo, para mostrar una nota de pié de página o comentario.
  • En una cita, para mostrar información sobre el autor y otros datos.


Atributos form

Se conoce como formulario a los mecanismos para enviar información por parte del usuario, a través de unos campos visuales de forma sencilla e intuitiva, que determinan la naturaleza del tipo de información que se va a enviar y como hacerla llegar a otro extremo donde se procesará esa información. El elemento "form" representa una sección de un documento que contiene controles interactivos para que el usuario envíe información a un servidor web.

Action

El atributo action en HTML se utiliza para indicar la dirección URL a la que se envían los datos de un formulario cuando se pulsa el botón de envío. Normalmente, los datos del formulario se envían a un archivo en el servidor cuando el usuario hace clic en el botón de envío.


Method

El atributo method se utiliza en HTML para definir cómo se envían los datos de un formulario al servidor web.

  • Método GET: Los valores enviados se agregan a la dirección indicada en el atributo action.
  • Método POST: Los valores se envian de forma separada.

Name

El atributo name en HTML se utiliza para identificar y organizar datos, y para referenciar elementos en JavaScript. Es compatible con varios elementos HTML, como los controles de formulario input, select, textarea y button.

Algunas de las funciones del atributo name son:

  • Identificar elementos y sus valores al enviar los datos de un formulario
  • Identificar un dato de un formulario en un servidor o base de datos
  • Dar un grupo específico a un conjunto de inputs de tipo checkbox o radio
  • Manejar la información del formulario desde JavaScript o desde el backend haciendo referencia al nombre


Atributos input

El atributo input en HTML se utiliza para crear campos de entrada en formularios, que permiten a los usuarios proporcionar y enviar información a una página web. Los campos de entrada son fundamentales para la interacción entre el usuario y el servidor, y para recopilar datos para procesos como registros, logins y otras transacciones.

El atributo input se utiliza de la siguiente manera:

  • Input de texto: Se utiliza para recibir pequeños strings de caracteres, como nombres, apellidos, fechas de nacimiento, etc. Es el campo de entrada de datos por defecto si no se indica ningún atributo type.
  • Área de texto: Se utiliza para inputs de texto largos, ya que permite múltiples líneas.
  • Atributo submit o image: Especifica el método HTTP que el navegador usará para enviar el formulario.
