Etiquetas de documentos

HTML

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo. Es decir, las etiquetas HTML son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML.


Head

Cuando hablamos de la etiqueta (head) de un documento HTML, muchas veces la denominamos cabecera HTML. Sin embargo, esta cabecera no es una parte visual de la página web, sino una parte de nuestro código HTML donde se incluyen ciertas etiquetas de metadatos, es decir, unas etiquetas que establecen ciertos datos que no tienen que verse necesariamente de forma visual.


Body

La otra parte principal de un documento HTML es la etiqueta (body). Todos los elementos visuales de una página se encuentran en el interior de la etiqueta "body", por lo que es una de las partes más importantes de una página web. Esta sección va inmediatamente después del cierre de la etiqueta (/head). Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.



Etiquetas de meta datos

Las etiquetas meta, también conocidas como metadatos, son fragmentos de código HTML que se utilizan para describir el contenido de una página web. Se colocan en el encabezado del código fuente, entre las etiquetas (html) y (body).

Los metadatos son importantes para los motores de búsqueda, los navegadores y los servidores. Los motores de búsqueda utilizan los metadatos para determinar el contenido de una página y ofrecerla a los usuarios. Una estructuración correcta de los metadatos puede mejorar el posicionamiento del sitio web

Meta

Quizás, una de las etiquetas HTML más populares que se colocan en el (head) de un documento HTML es la etiqueta (meta). Se trata de una etiqueta que nos permite indicar metadatos o información interna del documento HTML. Estos datos no se suelen mostrar visualmente en la página, sino que el navegador los lee para conocer información relevante y utilizarlos en situaciones específicas.

Por norma general, la etiqueta (meta) tiene los atributos name y content. Dependiendo del valor que le demos a name, tendremos que indicar un valor content correspondiente. Estas etiquetas se colocan en el interior de la etiqueta "head" de nuestro documento.


Style

la etiqueta style sirve para incluir estilos CSS que afecten sólo a la propia página HTML.

  • Insertar códigos CSS en el documento HTML.
  • Manipular características como fuente, color, diseño y otros aspectos visuales de las páginas web La etiqueta de es

Title

La etiqueta "title" sirve para indicar un título para nuestra página o documento HTML, y es obligatoria, por lo que cualquier documento HTML debería tener una etiqueta "title" (y sólo una, no deben existir varios). La etiqueta title es un código HTML que se utiliza para dar título a una página web. El título se muestra en la barra de título del navegador y en las páginas de resultados de los buscadores.

La etiqueta title es importante porque:

  • Proporciona información sobre el tema principal de la página a los usuarios y a los motores de búsqueda
  • Es el texto azul que se hace clic en los resultados de búsqueda
  • Tiene un impacto significativo en la optimización de motores de búsqueda (SEO)
  • Se utiliza para proporcionar contexto sobre un enlace compartido en las redes sociales

Base

Existe una etiqueta llamada "base" que puede utilizarse en la cabecera "head" del documento HTML. Esta etiqueta sirve para establecer una ruta base de los enlaces relativos utilizados en nuestro documento.


Link

La etiqueta "link" sirve para establecer relaciones con otros documentos, enlaces o archivos. De esta forma, el navegador o cualquier sistema capaz de leer código HTML puede saber que un documento está relacionado con otro (independientemente del formato que tenga) y así pueden establecerse ciertas relaciones. La etiqueta "link" se coloca en el encabezado de la página web, entre "head" "/head". Para enlazar a un recurso, se debe incluir el atributo href, que indica el destino del enlace.

Por ejemplo, para enlazar a una URL, se debe incluir href="url" en el elemento del enlace y poner la URL entre comillas.


Etiquetas de Agrupacion

Las etiquetas de agrupación en HTML son códigos que permiten agrupar elementos de un texto, como el título, la introducción, el contenido, entre otros.

P

HR

la etiqueta "hr" se utiliza para establecer una separación temática. Antes de continuar hay que hacer un alto aquí. El objetivo de la etiqueta "hr" es establecer una separación entre los contenidos que se encuentran antes de esta etiqueta y después, por lo que semánticamente sería más adecuado hablar de separación temática, en lugar de línea horizontal. Ten en cuenta que mediante CSS podemos cambiar el aspecto visual, para que no sea una línea sino un rectánngulo, un círculo, imágenes o eliminar completamente el aspecto visual, sin embargo, en todos estos casos, la separación temática seguirá existiendo.

La etiqueta "li" sirve para indicar los elementos internos de una lista HTML, ya sea no enumerada o lista enumerada (que veremos más adelante). Podemos añadir los ítems que deseemos:

Figure

La etiqueta "figure" en HTML se utiliza para representar contenido independiente, como imágenes, ilustraciones, diagramas, listas de códigos, entre otros. El contenido de la etiqueta "figure" está relacionado con el flujo principal de la página, pero su posición es independiente.

La etiqueta "figure" se utiliza en conjunto con la etiqueta "figcaption" para describir el contenido. La etiqueta "figcaption" se coloca como el primer o el último elemento secundario del elemento "figure". El texto de "figcaption" se muestra generalmente debajo de la imagen, video, audio, fragmento de código, tabla o información que contiene la ilustración "figure".


Figcaption

La etiqueta "figcaption" se utiliza en HTML para añadir una descripción o pie de foto a una imagen, video, audio, fragmento de código, tabla o información que contiene la ilustración "figure".

La etiqueta "figure" representa contenido gráfico, como una imagen, diagrama o gráfico, y la etiqueta "figcaption" se utiliza para complementarla. El texto de la etiqueta "figcaption" suele aparecer debajo de la imagen y se utiliza para ampliar o aclarar la información que se muestra en la figura.


Etiquetas Textuales

"a"

La etiqueta HTML "a" se utiliza para crear enlaces a otras páginas de internet, archivos, direcciones de correo, o cualquier otra URL. Es una de las etiquetas más importantes del lenguaje HTML y permite navegar por internet.

La letra "a" es la abreviatura de la palabra inglesa "anchor", que significa "ancla".

Para crear un enlace, se pueden utilizar los atributos "href" y "target":

  • "href" se utiliza para especificar la URL o ruta de destino del enlace.
  • "target" se utiliza para especificar si el enlace se abre en la misma página o en una pestaña nueva. Para ello, se le puede asignar el valor "self" o "_blank".

Span

La etiqueta HTML "span" se utiliza para dar estilo a texto o agrupar elementos en línea:

  • Dar estilo: Se puede usar para aplicar estilo a palabras o caracteres individuales. Por ejemplo, en una página de recetas, se puede usar para resaltar un ingrediente importante.
  • Agrupar elementos: Se puede usar para agrupar elementos en línea.

La etiqueta "span" es un elemento contenedor en línea genérico, lo que significa que el nombre de la etiqueta no indica nada sobre el contenido del elemento. En contraposición, un elemento HTML semántico tiene un nombre que describe su propósito, como "p" para párrafo.

La etiqueta "span" se diferencia de la etiqueta "div" en que "span" es un elemento en línea, mientras que "div" es un elemento de bloque. "Div" se usa para agrupar elementos HTML en un bloque.


Sub

La etiqueta HTML "sub" se utiliza para definir un fragmento de texto en subíndice, es decir, más abajo y generalmente más pequeño que el texto principal.


Sup

Por otro lado, la etiqueta "sup" se utiliza para definir un fragmento de texto en superíndice, es decir, medio carácter por encima de la línea normal y, a veces, en una fuente más pequeña.

La etiqueta "sup" en HTML se utiliza para mostrar un fragmento de texto en superíndice, es decir, más alto y generalmente más pequeño que el resto del texto. Se usa comúnmente en contextos matemáticos y científicos, como en referencias, notas al pie, exponentes, abreviaturas francesas, etc.


Cite

La etiqueta "cite" en HTML se utiliza para indicar la fuente o el autor de un texto citado.

Las etiquetas "cite" y "/cite" son obligatorias. En navegadores basados en Mozilla, el contenido de "cite" se muestra en cursiva.

La etiqueta "cite" se utiliza para citar obras creativas, como libros, películas, canciones, pinturas, obras de arte, juegos, páginas web, tuits, o posts en Facebook. Es importante tener en cuenta que la etiqueta "cite" no reemplaza las buenas prácticas de citación en la escritura. Para marcar el texto de las citas, se pueden usar los elementos "q" (en línea) y blockquote (en bloque).


Etiquetas Semanticas

Las etiquetas semánticas de HTML son elementos de código que definen el significado del contenido de una página web y ayudan a estructurarlo. Su objetivo es que el contenido sea claro y legible para los usuarios y los motores de búsqueda.

Las etiquetas semánticas ayudan a:

  • Organizar el contenido de una página.
  • Mejorar la accesibilidad para personas con discapacidades.
  • Proteger el sitio web para el futuro, ya que es más probable que los elementos semánticos sean compatibles y actualizados.

Nav

La etiqueta "nav" en HTML se utiliza para crear una sección de navegación en una página web. Su función es agrupar los enlaces de navegación, ya sean menús, barras de navegación, enlaces a otras páginas, o cualquier otro elemento que ayude a los usuarios a moverse entre las secciones de la página.

Es posible tener varios elementos "nav" en nuestro sitio, ya que por ejemplo, quizás necesitemos una segunda o tercera agrupación de enlaces de navegación porque visualmente están separados de los demás bloques. En ese caso, se pueden utilizar nuevos elementos "nav" para agrupar enlaces de navegación.


Section

Mediante la etiqueta "section" es posible crear apartados para una web, de modo que las mantenemos separadas. Muchas veces no suele estar clara que etiqueta utilizar, así que vamos a simplificar y generalizar los pasos:


Article

La etiqueta HTML "article" se utiliza para agrupar información muy relevante de una página o documento concreto. Lo primero, es pensar en nuestro documento y reducirlo a la mínima expresión en una lista. Por ejemplo, si pensamos en la página principal de un medio de prensa que muestra los últimos 5 artículos publicados.


Header

La etiqueta "header" en HTML se utiliza para definir la sección de encabezado de una página web. Es el primer elemento que aparece en el código HTML y contiene información importante sobre la página, como el título, la descripción y las palabras clave.

El encabezado es crucial para mejorar la experiencia del usuario y la navegación en la página. También tiene un impacto en el SEO ya que los motores de búsqueda utilizan la información del encabezado para indexar y clasificar el contenido de la página. El encabezado puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes.


Main

La etiqueta "main" es una etiqueta de agrupación general que debe utilizarse para representar el contenido dominante y principal del documento. De forma habitual, esta etiqueta suele ser elemento hijo de etiquetas como "body", "div", "form" o webcomponents.

La etiqueta "main" funciona de manera similar a una etiqueta "div". Se puede manejar con CSS para visualizarla como se quiera, y tiene atributos como "class" y "id".


Footer

De la misma forma que hemos visto la etiqueta "header", tenemos la etiqueta "footer". Mientras que la primera se utiliza para indicar cabeceras, esta última se utiliza para agrupar los elementos a pie de página o elemento. La forma de utilizarla, es exactamente igual que la etiqueta "header", incluso utilizándola como hija de un elemento semántico.

Su función es:

  • Indicaar el final del sitio web.
  • Contener información que no es fundamental para entender el contenido principal, pero que es importante para ofrecer una experiencia de usuario completa.
  • Facilitar la navegación y enlazar información para que los usuarios la encuentren fácilmente.
